這是 C++ 遊戲《尋傘記：政大山下篇》的 整個 repo 知識圖譜——把每一個檔案、OO 概念、設計模式，和它們之間的相依關係（#include、類別繼承、模式落點）全部畫成一張可互動的地圖。

圓點 = 一個程式檔（顏色＝領域）
星形 = 設計模式 / OO 概念
連線 = 相依

怎麼用：

• 滑鼠拖曳、滾輪縮放
• 滑鼠移到節點上 → 高亮相鄰、看提示
• 點節點 → 右側看介紹、原始碼與逐檔說明連結
• 雙擊節點 → 直接開逐檔說明
• 左側欄可搜尋檔名 / 類別、依領域過濾
