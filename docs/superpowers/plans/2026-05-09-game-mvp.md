# 《尋傘記》Day 1 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the OOP_Raylib_Lab MVP that satisfies all 7 Assignment 5 rubric items by end of Day 1 (2026-05-09), using color-block placeholders for visuals.

**Architecture:** Inheritance tree per `系統架構與UML分析：尋傘記.md` — `GameObject` → {`Character` → `Player`, `Item` → `TransparentUmbrella` → `TrueUmbrella`}. `GameObjectFactory` produces concrete subtypes by enum (Factory Method). `EventBus` decouples Item events from UI rendering (Observer). Main loop iterates `std::vector<std::unique_ptr<GameObject>>` calling `Update`/`Draw`. **No `LoadTexture` calls in Day 1** — all visuals are `DrawRectangle` colored blocks.

**Tech Stack:** C++17, Raylib 5.5 (FetchContent), CMake 3.14+, doctest (single-header, FetchContent).

**Hard rules (CLAUDE.md §"Hard architectural rules"):**
1. `Player.h` / `Player.cpp` MUST NOT `#include` any concrete umbrella header.
2. `Item*` and `*Umbrella*` files MUST NOT call raylib draw primitives directly — they emit events via `EventBus`.
3. `MapManager` (or `main.cpp`'s update loop) MUST NOT `delete` GameObjects mid-iteration; mark `isActive=false` and sweep at end of frame.

**Out of scope for Day 1** (Day 2-3 work):
- `FragileUmbrella`, `ProfessorTrapUmbrella`, `CursedUmbrella` (only `TrueUmbrella` in Day 1)
- `NPC` class
- `SemesterStateMachine`
- Real sprite textures (Day 3)

## Task 5: EventBus (Observer pattern) + tests

> **Order rationale:** EventBus has no dependencies, so it ships first. TransparentUmbrella (Task 6) needs `EventBus.h` for its `Draw()` event emission, so writing EventBus first lets every commit pass `cmake --build`.

**Files:**
- Create: `include/EventBus.h`
- Create: `src/EventBus.cpp`
- Create: `tests/test_eventbus.cpp`

- [ ] **Step 1: Write EventBus.h**

```cpp
#pragma once
#include "raylib.h"
#include <functional>
#include <string>
#include <unordered_map>
#include <vector>

enum class EventType {
    RenderRequested,
    UmbrellaClaimed,
    KarmaChanged,
    ShowMessage,
};

struct Event {
    EventType type;
    Vector2 position;
    Color color;
    std::string text;
};

class EventBus {
public:
    using Handler = std::function<void(const Event&)>;

    static EventBus& Instance();

    void Subscribe(EventType type, Handler handler);
    void Publish(const Event& event) const;
    void Clear();

private:
    EventBus() = default;
    std::unordered_map<EventType, std::vector<Handler>> handlers_;
};
```

- [ ] **Step 2: Write EventBus.cpp**

```cpp
#include "EventBus.h"

EventBus& EventBus::Instance() {
    static EventBus instance;
    return instance;
}

void EventBus::Subscribe(EventType type, Handler handler) {
    handlers_[type].push_back(std::move(handler));
}

void EventBus::Publish(const Event& event) const {
    auto it = handlers_.find(event.type);
    if (it == handlers_.end()) return;
    for (const auto& h : it->second) h(event);
}

void EventBus::Clear() {
    handlers_.clear();
}
```

## Task 9: Wire main.cpp game loop

**Files:**
- Modify: `src/main.cpp` (replaces existing 8-line stub)

- [ ] **Step 1: Replace main.cpp**

```cpp
#include "raylib.h"
#include "GameObject.h"
#include "GameObjectFactory.h"
#include "Player.h"
#include "EventBus.h"
#include <memory>
#include <vector>
#include <iostream>
#include <algorithm>

int main() {
    InitWindow(800, 450, "Lost Umbrella - MVP");
    SetTargetFPS(60);

    // Subscribe a console-print handler to demonstrate UI/Data separation.
    // In production, a UIManager would render dialogs from these events.
    EventBus::Instance().Subscribe(EventType::ShowMessage,
        [](const Event& e) { std::cout << "[UI] " << e.text << '\n'; });
    EventBus::Instance().Subscribe(EventType::UmbrellaClaimed,
        [](const Event& e) { std::cout << "[Game] Claimed: " << e.text << '\n'; });
    EventBus::Instance().Subscribe(EventType::RenderRequested,
        [](const Event& e) {
            DrawRectangle((int)e.position.x, (int)e.position.y, 20, 20, e.color);
        });

    std::vector<std::unique_ptr<GameObject>> objects;
    objects.push_back(GameObjectFactory::Create(ObjectType::Player, {400, 225}));
    objects.push_back(GameObjectFactory::Create(ObjectType::TrueUmbrella, {200, 100}));

    // Player* lookup for collision resolve
    Player* player = dynamic_cast<Player*>(objects.front().get());

    while (!WindowShouldClose()) {
        float dt = GetFrameTime();

        // Update phase
        for (auto& obj : objects) {
            if (obj && obj->IsActive()) obj->Update(dt);
        }

        // Pickup interaction: player presses E while overlapping an item
        if (IsKeyPressed(KEY_E) && player) {
            for (auto& obj : objects) {
                if (!obj || !obj->IsActive() || obj.get() == player) continue;
                Rectangle pHit{player->GetPosition().x, player->GetPosition().y, 24, 24};
                if (obj->CheckCollision(pHit)) {
                    obj->Interact(player);
                }
            }
        }

        // Draw phase
        BeginDrawing();
        ClearBackground(RAYWHITE);
        for (auto& obj : objects) {
            if (obj && obj->IsActive()) obj->Draw();
        }
        DrawText("WASD: move    E: pick up", 10, 10, 16, DARKGRAY);
        if (player) {
            DrawText(TextFormat("karma: %d   umbrella: %s",
                player->GetKarma(), player->HasUmbrella() ? "yes" : "no"),
                10, 30, 16, DARKGRAY);
        }
        EndDrawing();

        // End-of-frame sweep: deferred deletion (CLAUDE.md hard rule #3)
        objects.erase(
            std::remove_if(objects.begin(), objects.end(),
                [](const std::unique_ptr<GameObject>& o) {
                    return !o || !o->IsActive();
                }),
            objects.end());
        if (player && std::find_if(objects.begin(), objects.end(),
                [player](const std::unique_ptr<GameObject>& o) { return o.get() == player; })
            == objects.end()) {
            player = nullptr; // player was swept (shouldn't happen in MVP, but stay safe)
        }
    }

    CloseWindow();
    return 0;
}
```
