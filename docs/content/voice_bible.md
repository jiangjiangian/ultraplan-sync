# 《尋傘記》5 NPC Voice Bible

## 如何使用本檔

本檔是 Ch2 / Ch3 / Ch4 與三個結局的**對白基準書**。
在為任何章節撰寫 NPC 台詞前，請先確認：

1. 口頭禪與語氣是否落在本檔「三、口頭禪 / 語氣樣本」的範圍內。
2. Karma 高低對應「五、Karma 高/低時的語氣差」，而非自行發明分支。
3. Voice arc 章節狀態對應「四、四章 voice arc」——同一 NPC 在不同章節情緒有移動，不是平坦的單一面孔。
4. 任何 ripple flag（如 `Flag_HelpedTA_Ch1`、`Flag_PromisedVictim`）在對白中要有可感知的落地，而非只在程式層存在。
5. 每行對白 **≤ 28 全形字**，不得超出，對白框會 overflow。

## NPC：西裝學長

### 一、人設一句話

商學院的菁英人設愛好者，用人脈當貨幣的話術販子——對你熱心的背後永遠有一件小事要麻煩你。

### 二、人設細節

- **身分**：大三或大四，商學院（財金或企管系）。
  > 建議補設定，待覆核：可設定為財金系大四，正在跑校內企業實習面試。
- **外型印象**（供 sprite 生成參考）：深藍或炭灰細條紋西裝，白襯衫，領帶微歪。頭髮用髮蠟稍微整過，但右側有一撮翹起來。皮鞋是黑色的，但鞋底沾了泥。整體感：「剛面試完、還沒換回便服、但疲態已經漏出來了」。
- **最在乎的事**：在對的人面前顯得有用、有消息、有管道。被人看穿只是在吹牛比被拒絕更讓他難受。
- **最害怕的事**：面試失敗、被同學發現他的「管道」是虛的、被助教當場點名。
- **與主角的關係軸**：工具人 ↔ 恩人。Ch1 是把玩家當跑腿的工具人；若玩家選擇替他圓場而非斥責，他記得這份情，Ch4 時反過來成為玩家的情報提供者（`Flag_HelpedSenior = true` 路徑）。

### 三、口頭禪 / 語氣樣本

**典型開場白（不超過 28 字）**

- 「我有個管道，幫你喬一下應該沒問題。」
- 「你要不要先聽我說？我這邊有資訊。」
- 「啊不是啦，你聽我解釋——」
- 「我認識一個學長，他那邊說……」
- 「只是順手的小事，怎麼這樣看我？」

**不會說的話（負面樣本）**

- 直白道歉（他會迂迴轉移話題，不會直說「我錯了，對不起」）
- 示弱到令人同情的程度（最多裝一下無辜，不會崩潰哭泣）
- 使用學術或文謅謅的詞彙（他的語氣是業務感的輕鬆，不是書卷氣的嚴謹）
- 「我不認識任何人」——他永遠在說認識誰，哪怕是編的

### 四、四章 voice arc

- **Ch1（加退選）**：剛結束面試的疲倦推銷者。還在試探玩家，話術是包裝過的，傲氣比不安多。口吻親切但帶有居高臨下的輕描淡寫。
- **Interlude（四維道市集）**：出場可有可無。若出場，是在市集附近滑手機等人，對玩家的態度因 Ch1 旗標而分歧——`Flag_ScoldedSenior = true` 時，他遠遠看到你就側身走開。
- **Ch2（期中考）**：不在中正圖書館；這章他缺席。若設計師想加回呼，可設定他出現在自習室門口抱怨考試範圍太廣——但非主線出場。
  > 建議補設定，待覆核：可設計為 Ch2 可選對話，在行政大樓前遇到，僅一兩行，維持存在感。
- **Ch3（運動會）**：在啦啦隊加油區邊緣，不是真的在幫本系加油，是在認識其他系的人。口氣輕鬆得像是沒有任何考試壓力。若 `Flag_HelpedSenior = true`，他會主動打招呼並給一條物物交換鏈的提示。
- **Ch4（期末考）**：只在 `Flag_HelpedSenior = true` 路徑出現。站在行政大樓門口，眼神比 Ch1 少了一點油滑，多了一點真誠。這次他主動找你，不是要你幫他。
- **Ending A 看到的他**：在四維道遠端，西裝換成了普通外套，手機放進口袋，跟你點頭示意。說了一句話就走了，少了很多廢話。
- **Ending B 看到的他**：根本不會出現。你成了他；他已經消失在人群裡了。
- **Ending C 看到的他**：看到你手上的醜綠傘，忍不住笑了一聲，搖搖頭走開，沒有評語。

### 五、Karma 高/低時的語氣差

> Karma 刻度提醒：起始 50，×0.6 累積型。「Karma > 70」代表玩家在多個章節中持續做出正向選擇，非一次善行即達；「Karma < 30」代表多次冷漠或投機積累，非單一惡行。

- **Karma > 70**（玩家長期表現正直，學長感受得到）：
  - 「你這個人……還蠻難讓我說謝謝的，但這次是真的。」
  - 「行吧，我知道你一直都在幫人。我欠你一個。」
- **Karma 50 中性**（平均水準，學長按例待你）：
  - 「我有個管道，幫你喬一下應該沒問題。」
  - 「就是個小忙嘛，你要不要試試看？」
- **Karma < 30**（玩家行事冷漠或投機，學長感覺不對勁）：
  - 「……你最近做的事，我有點耳聞。」
  - 「算了，各憑本事吧。我管道也不是給每個人用的。」

### 六、與其他 NPC 的關係

- **苦主**：Ch1 正是學長拿走了苦主的傘（主線已定）。兩人互不認識，學長並非刻意針對苦主，只是順手拿了傘架上的傘。
- **助教**：學長因為 ProfessorTrapUmbrella 間接讓助教變成 Ch1 的追逐 NPC，但兩人也不相識。學長不知道那把傘是教授的。
- **學霸**：互不熟，可能在圖書館見過面，但學長不會在圖書館待太久。
- **福利社阿姨**：學長是熟客（常買咖啡），阿姨叫得出他，但不知道他的名字，只叫「那個西裝同學」。

## NPC：學霸

### 一、人設一句話

全校自習時數最高的讀書機器，對人冷淡不是因為傲慢，而是因為他腦子裡有一半時間都在模擬考題。

### 二、人設細節

- **身分**：大二或大三，理工或法律系（精確系所未定）。
  > 建議補設定，待覆核：可設定為法律系大三，備考律師高考，解釋他為何在任何天氣都執著於找安靜角落讀書。
- **外型印象**（供 sprite 生成參考）：大學T或帽T，外加一件拉鍊外套。永遠背著一個巨大的後背包，裡面裝了至少三本厚重講義。頭髮沒有特別整理，眼睛有淡淡的黑眼圈。不戴眼鏡，但眯眼看東西的習慣讓他看起來像戴著。
- **最在乎的事**：效率。他對任何能打斷他讀書節奏的事情都有輕微的不耐煩——包括回答你的問題。但他是公正的，不會因為不認識你就敷衍你。
- **最害怕的事**：考試表現低於預期；浪費時間在無意義的事情上。
- **與主角的關係軸**：中立的資訊節點。Ch1 給了玩家有用情報（對話 b，追蹤學長）；Ch2 卻是拿傘的那個人，從資源變成障礙，再到補贖。

### 三、口頭禪 / 語氣樣本

**典型開場白（不超過 28 字）**

- 「你說什麼？快一點，我等一下還要背書。」
- 「去問管理員，她知道的比我多。」
- 「找到了嗎？你有去試，這樣就夠了。」
- 「我沒有在浪費時間，你要問什麼？」
- 「喔，那個——你等一下，我把這段讀完。」

**不會說的話（負面樣本）**

- 長篇的情感傾訴（他若有情緒，只會露出一點縫，不會開閘大倒）
- 炫耀成績（成績對他是理所當然，不是拿來說嘴的東西）
- 閒聊和天氣（除非天氣影響到讀書計畫）
- 「你好可憐哦」之類有同情意味的句子（他的暖意是行動型的，不是語言型的）

### 四、四章 voice arc

- **Ch1（加退選）**：路邊資訊站。他有用，但無情感投入。給出資訊後立刻想結束對話。這是他最「機械」的狀態——還沒進考試地獄，情緒平台。
- **Interlude（四維道市集）**：幾乎不出現。若有，是靠著攤位角落在看書，對市集的熱鬧一點反應都沒有。如果玩家主動靠近，他只說：「現在還不算雨季，好好利用。」
- **Ch2（期中考）**：本章主角對應 NPC，是「傘被學霸拿走」的當事人。他拿傘是無意識的（讀書讀到當機，下意識抓了放在桌旁的傘就走出去）。此時他的語氣是遊魂式的，說話答非所問，但執行力仍在。
- **Ch3（運動會）**：不在操場。在圖書館的某個角落繼續讀書，完全無視運動會。若玩家找到他，他會說：「那種場合我不去，吵。」
  > 建議補設定，待覆核：可設計為 Ch3 可選支線——玩家若找到學霸並說服他去看大隊接力，可解鎖一條情報捷徑，減少物物交換步驟。
- **Ch4（期末考）**：他比任何章節都更短暫，但質地最不同。他早在傘還沒丟失前就在考場外等待，玩家若路過，他說了一句讓人意外的話——幾乎是在關心玩家。這是補贖。
- **Ending A 看到的他**：在中正圖書館門口，陽光出現了，他瞇著眼睛看了看天空，然後低頭繼續翻講義。只說：「出太陽了，奇怪。」
- **Ending B 看到的他**：圖書館燈還亮著，但已經不是他了。你只看到一個拿著透明傘的背影走進門，認不出臉。
- **Ending C 看到的他**：看到你的醜綠傘，思考了三秒，說：「防混用。合理決策。」然後轉身離開。

### 五、Karma 高/低時的語氣差

- **Karma > 70**（玩家一路行事正直，學霸雖然不在意 karma，但態度略微開放）：
  - 「你每次都這樣嗎？幫別人然後繼續走。」
  - 「……行，我欠你一個。下次優先回答你的問題。」
- **Karma 50 中性**（預設狀態，他按照資訊量回應你）：
  - 「去問管理員，她知道的比我多。」
  - 「找到了嗎？你有去試，這樣就夠了。」
- **Karma < 30**（玩家行事投機，學霸有感知但不評論，只是更短促）：
  - 「你問我幹嘛，去問別人。」
  - 「（沉默三秒）沒什麼好說的。」

### 六、與其他 NPC 的關係

- **西裝學長**：互不相識，可能在圖書館座位擦肩而過，但學霸不記得那個人。
- **助教**：學霸偶爾去助教輔導時間問問題，關係是純業務性質，不熟。
- **福利社阿姨**：偶爾去買提神飲料。阿姨叫得出他是「那個靠窗坐的同學」，但不知道他叫什麼。
- **苦主**：不認識。可能在傘架旁擦肩而過，但學霸不記人臉。

## NPC：助教

### 一、人設一句話

外表是流程守門人，內心是永遠在燒的行程表——他的嚴肅是一層薄殼，底下是快崩掉的焦慮。

### 二、人設細節

- **身分**：研究所碩二，擔任某科目（數理或社科，精確課程未定）的助教。
  > 建議補設定，待覆核：可設定為資訊系或統計系碩二，正在同時跑論文與助教工作，解釋他為何在 Ch4 三天沒睡。
- **外型印象**（供 sprite 生成參考）：戴眼鏡，深色休閒外套或圓領毛衣，手上永遠夾著一疊 A4 紙或資料夾。臉上有慢性疲勞的印記，但站姿仍然筆直。嘴巴抿緊是他的預設表情。眼鏡有時候會滑下來，然後他習慣性地用中指推上去。
- **最在乎的事**：把流程做對，不出包。他的嚴格不是針對人，而是針對程序。
- **最害怕的事**：教授因為他的失誤而被波及；資料或成績出問題；在公開場合失態。
- **與主角的關係軸**：程序的鏡像。Ch1 是冷漠守門人；若玩家幫助他（`Flag_HelpedTA_Ch1 = true`），他在 Ch4 雖然已經崩潰，但留了一絲人味——這是這個角色唯一的弧線出口。

### 三、口頭禪 / 語氣樣本

**典型開場白（不超過 28 字）**

- 「同學，流程就是這樣，沒有例外。」
- 「填表，等通知，就這樣。」
- 「……這不在我的業務範圍。」
- 「你先稍等，我手上還有東西。」
- 「（快步走過）我現在很忙，之後再說。」

**不會說的話（負面樣本）**

- 情緒化的爆發（他就算崩潰也是靜態的崩潰，說話速度變慢、語氣乾扁，不會大吼）
- 隨興的閒聊（他對「閒」字過敏）
- 替人情緒勞動（他不會說「你看起來很辛苦哦」，最多是「你去休息一下」，句號結尾）
- 空話承諾（他說得到的事他會做，說不到的他不說）

### 四、四章 voice arc

- **Ch1（加退選）**：官僚防火牆。公事公辦，不多給一個字。但若玩家幫他找回被風吹走的申請書，他會露出一點人性的縫——感謝的方式是提供情報，不是溫言。這是他最「防禦」的狀態。
- **Interlude（四維道市集）**：不出現。他在辦公室整理資料。
- **Ch2（期中考）**：漣漪效應出場點。若玩家 Ch1 拿了 `ProfessorTrapUmbrella` 並且在 Ch2 路過中正圖書館門口，助教在此認出玩家，觸發 karma -10，並給出一段冷淡但不殘忍的警告。若沒有此旗標，助教在 Ch2 只是一個路過的背景 NPC，有一行台詞：「期中考週，圖書館人多，大家加油。」
- **Ch3（運動會）**：在操場邊緣架桌子登記活動，依然是流程守門人，但語氣裡有停課的輕鬆感——極微量，但存在。
- **Ch4（期末考）**：本章的人性頂點與崩潰點。三天沒睡，拿了玩家的傘（無意識的，因為他的傘在辦公室忘了帶）。若 `Flag_HelpedTA_Ch1 = true`，他認出你，說出一段帶有歉意的話，語氣還是乾，但字裡行間有東西。若旗標為 false，他只是一個抱著考卷跑路的陌生人。
- **Ending A 看到的他**：在正門附近，手上沒有考卷了，第一次看起來不那麼趕。跟你說「歐趴糖，你的」，遞過來，然後轉身。
- **Ending B 看到的他**：不出現。只有一張貼在告示欄的「助教輔導時間異動公告」，日期是空白的。
- **Ending C 看到的他**：看到你的醜綠傘，眉毛動了一下，說：「防偷。好主意。」然後繼續走。

### 五、Karma 高/低時的語氣差

- **Karma > 70**（玩家一路正直，助教對你的判斷多了一分信任）：
  - 「加退選的事我幫你備註了。不保證，但會試。」
  - 「你這學期……做了蠻多的。我有看到。」
- **Karma 50 中性**（預設程序員模式）：
  - 「填表，等通知，就這樣。」
  - 「這不在我的業務範圍。」
- **Karma < 30**（玩家多次不配合或投機，助教態度更冷，但不失禮貌的底線）：
  - 「同學，規定就是規定，請配合。」
  - 「（沒看你，繼續整理資料）下一個。」

### 六、與其他 NPC 的關係

- **西裝學長**：學長因 ProfessorTrapUmbrella 間接讓助教陷入 Ch1 的混亂，但兩人並不認識。
- **學霸**：學霸偶爾出現在助教輔導時間，是他已知的臉孔之一，但關係仍是業務性的。
- **福利社阿姨**：助教偶爾外帶咖啡，阿姨認識他，知道他愛大杯黑咖啡、不加糖。
- **苦主**：不相識，但助教在 Ch1 的傘架附近有注意到有人在哭，他沒有上前。

## NPC：福利社阿姨

### 一、人設一句話

樂活小舖的靈魂人物，用碎念包住溫度，知道每個常客喜歡什麼飲料的那種人。

### 二、人設細節

- **身分**：樂活小舖店主，中年女性，在政大服務多年。
  > 建議補設定，待覆核：可設定她在此工作超過十年，見過至少三屆以上的學生循環，解釋為何她對每個學期的「加退選混亂」說起來像在背劇本。
- **外型印象**（供 sprite 生成參考）：圍裙是藍色或綠色的（不是制服感，是家庭感），頭髮往上盤，偶爾有一縷掉下來。笑容帶著一點疲憊，但是真心的。手上通常在做事：整理貨架、抹桌子、核對帳單。
- **最在乎的事**：店裡的東西好好賣出去，客人身體好；還有那個月的帳算清楚。
- **最害怕的事**：進貨到一半突然下大雨（貨會濕）；學生有急事又買不起東西（會想賒帳給他們，但她說了不能再這樣了）。
- **與主角的關係軸**：生活的錨點。她不給任務，但她的店是玩家最安全的避雨點，她的碎念是最真實的校園體溫計。Ending C 的路徑種子就在她手裡。

### 三、口頭禪 / 語氣樣本

**典型開場白（不超過 28 字）**

- 「哎呀同學，淋這麼濕進來喔！」
- 「加退選又在搶了是嗎，年年都這樣。」
- 「今天有備熱的，暖暖身子。」
- 「要什麼你說，我幫你找。」
- 「你上次來是……哦，加退選那週。記得你。」

**不會說的話（負面樣本）**

- 說教學生的道德（她碎念，但不是教訓型的碎念；她不說「你應該要……」，而是「你要不要……比較好」）
- 對學生的困境冷漠（就算她在忙，她不會完全無視一個明顯很慘的學生）
- 用正式的語氣說話（沒有「請問您要……」這種句型）
- 假裝知道所有校園資訊（她知道的是人，不是系統）

### 四、四章 voice arc

- **Ch1（加退選）**：親切熱鬧。她多備了貨因為知道今天人多，語氣裡有一種「我見過這個」的從容。醜綠傘的 Ending C 種子在這章埋下。
- **Interlude（四維道市集）**：本章主要出場點之一。她在樂活小舖門口擺了一個小攤位，賣暖暖包和防水噴霧。碎念增多，但語氣裡有節慶感——她也覺得市集比平常有趣。
- **Ch2（期中考）**：語氣降了一個音量。她知道考試週大家壓力大，少說笑話，多給實用的。備品換成熱飲和護眼貼。對玩家說話時有一種「你沒事吧」的眼神。
- **Ch3（運動會）**：停課所以店裡比較冷清，但她反而顯得更輕鬆。去了一半的運動會然後提早回來。說話間會提到她看到了哪個系的大隊接力很精彩。
- **Ch4（期末考）**：最疲倦的狀態，但她還是在。備品換回黑咖啡和護腕（給需要長時間寫考卷的學生）。對玩家的語氣多了一點「快要結束了」的感傷。
  > 若 `Flag_BoughtCoffeeForAuntie_Ch1 = false`（玩家在 Ch1 沒有買咖啡給她），她在 Ch4 提供的情報較間接——她不會主動說助教往哪跑，但會說「最近那個常來的助教好像很趕」。
- **Ending A 看到的她**：在門口擺椅子吹涼風，暑假的第一天，店裡客人很少。她看到你撐著傘走過，喊了一聲「加油哦！」
- **Ending B 看到的她**：不直接出現。但店裡的架上，多了一把有別人名字的透明傘放在失物招領盒旁。
- **Ending C 看到的她**：是這個結局的見證人。她看著你抱著那把醜綠傘走出去，說了最後一句碎念。

### 五、Karma 高/低時的語氣差

- **Karma > 70**（玩家一路幫助別人，阿姨感覺得到你是個好學生）：
  - 「你這個同學哦，每次來都是幫別人買東西。」
  - 「下次加退選記得早點來，我幫你留一個熱的。」
- **Karma 50 中性**（常客互動，不冷不熱但有溫度）：
  - 「今天有備熱的，暖暖身子。」
  - 「要什麼你說，我幫你找。」
- **Karma < 30**（玩家給阿姨的感覺是「這個同學有點奇怪」，語氣保持熱情但收斂一點）：
  - 「……你還好嗎同學？臉色很差耶。」
  - 「東西拿好，錢算清楚，慢走哦。」

### 六、與其他 NPC 的關係

- **西裝學長**：熟客，阿姨叫他「那個西裝同學」，知道他喜歡拿鐵加奶精。
- **學霸**：偶爾來買提神飲料，阿姨叫他「那個靠窗的同學」，知道他要無糖黑咖啡。
- **助教**：常客，阿姨知道他要大杯黑咖啡不加糖，也注意到他最近臉色越來越差。
- **苦主**：她見過苦主在雨天蹲在傘架旁，有想過要出去問問，但當時店裡有客人。

## NPC：苦主

### 一、人設一句話

傘是她的，委屈也是她的，但她把這份委屈壓得很低——直到有人讓她覺得被記得，她才會讓你看到。

### 二、人設細節

- **身分**：大一或大二，系所不定。
  > 建議補設定，待覆核：可設定為大一下學期，剛度過入學適應期，這是她第一次在政大遇到傘被拿走的事，解釋她的無措感尚未變成「習慣了」。
- **外型印象**（供 sprite 生成參考）：外套的帽子拉起來，雨水打濕了肩膀那一塊。蹲著或靠著牆站著，不佔太多空間的姿態。書包放在地上，傘架旁邊。眼睛是往下看的，不是在哭，但眼眶有點紅。
- **最在乎的事**：那把傘（她媽買給她的，旅遊帶回來的）；以及在陌生的校園裡被人認真對待這件事。
- **最害怕的事**：再一次開口求助然後沒有人理。上一次已經讓她說不出口了。
- **與主角的關係軸**：道德的試金石。她是最先出現的 NPC，也是整個故事的「起因人」。Ch1 她是弱的；Ch3 / Ch4 她是記得的。若玩家背棄了承諾（`Flag_PromisedVictim = true` 但沒有還傘），Ch3 的她不是憤怒——她是那種讓你比憤怒更難受的失望。

### 三、口頭禪 / 語氣樣本

**典型開場白（不超過 28 字）**

- 「……我的傘不見了。」
- 「你也是被偷的嗎？」
- 「沒關係，我再想想辦法。」
- 「謝謝你……你要小心那個學長。」
- 「（沉默，然後輕聲）你還記得嗎？」

**不會說的話（負面樣本）**

- 激烈的控訴或當場爆發（她的記恨是靜態的、長尾的，不是當面發作型的）
- 對玩家冷嘲熱諷（她不會，她只會沉默或說「沒關係」；但那個「沒關係」比任何話都重）
- 主動要求玩家做任何事（她提出的請求都是被動被問出來的，不是主動推銷的）
- 過度依賴或黏著（她不會一直出現，她出現的每次都是有重量的）

### 四、四章 voice arc

- **Ch1（加退選）**：最脆弱的狀態。她剛失去傘，還沒整理好情緒。說話斷斷續續，沒有方向，靠著你問才能說出有用的資訊。她在乎那把傘不只是因為錢。
- **Interlude（四維道市集）**：不出現在市集。她在某個角落自己讀書，也許在避雨。
  > 建議補設定，待覆核：可設計為 Interlude 短暫遭遇，若玩家主動找她，她說「傘的事……謝謝你有幫我」（若 `Flag_PromisedVictim = true`），玩家可見到她情緒稍微穩一點。
- **Ch2（期中考）**：不在主線。她在圖書館某個角落備考，有黑眼圈，傘換成了借來的或塑膠袋。若玩家偶然遇到，她說：「你還在找啊？」語氣裡有一點「你比我執著」的意外。
- **Ch3（運動會）**：漣漪效應出場點。若 `Flag_PromisedVictim = true` 而傘尚未還給她，她在操場邊緣看到玩家，不會主動靠近，但玩家靠近時她的台詞不再是「你也是嗎」，而是那種很輕的「……你還沒找到」。這行台詞是 karma 的隱性扣點。
- **Ch4（期末考）**：最終的出場。若傘歸還（且玩家 karma 足夠），她出現在正門旁，說了一句完整的感謝。若玩家走 Ending B 路徑，她的台詞只有一行，她不知道那個人是你了。
- **Ending A 看到的她**：在正門附近，撐著那把找回來的傘，看到你，說：「找到了。謝謝你還記得。」
- **Ending B 看到的她**：她在傘架旁站著，像 Ch1 剛開始那樣。不是她的傘又不見了，而是她看到了某個人拿走的背影，和那個人一模一樣——但她不知道那個人是你。
- **Ending C 看到的她**：看到你手上的醜綠傘，愣了一下，然後輕聲說：「這樣以後就不會弄錯了。」說完轉身。

### 五、Karma 高/低時的語氣差

- **Karma > 70**（玩家真心承諾且履行，她對你的信任是真實的）：
  - 「你說過你會找，你真的找了。」
  - 「謝謝你……我沒想到有人記得。」
- **Karma 50 中性**（玩家有承諾但行動普通，她還在等但不是絕望）：
  - 「你還在找啊？……謝謝你有去試。」
  - 「沒關係，你盡力就好。」
- **Karma < 30**（玩家冷漠或背棄承諾，她不說指責的話，但那句「沒關係」說得像是「我知道了」）：
  - 「……沒關係。」
  - 「（低頭，拉了一下帽子，沒有再說什麼）」

### 六、與其他 NPC 的關係

- **西裝學長**：是傘被學長拿走的當事人，但她只知道「穿西裝的學長」，不認識這個人。
- **助教**：不相識。在行政大樓附近見過，沒有交集。
- **學霸**：不相識。
- **福利社阿姨**：阿姨在 Ch1 的雨天見過她蹲在傘架旁，印象模糊。若玩家在阿姨面前提到苦主，阿姨說：「那個蹲在外面的女同學？她沒事吧？」

對照 `chapter1.md` 已有對白，以下為建議關注點（不建議改，但接下來寫 Ch2+ 時應留意）：

- **助教 (a) 段語氣**：目前 Ch1 的助教台詞是純嚴肅（「同學，加退選截止了，現在不受理」）。專案設定他「表面嚴肅，實際慌張」。Ch1 現有對白尚未讓「慌張」的底層浮現；建議從 Ch2 或 Ch3 開始讓慌張感滲出，Ch4 再全揭。若希望 Ch1 就埋線，可在 (c) 段（助教認出 ProfessorTrapUmbrella 後）讓他說話的語速感變快，例如多一個破折號或刪節號。
